# Interactive Superstore Sales Dashboard | R Shiny

# Project Overview
The Interactive Superstore Sales Dashboard is an interactive business intelligence dashboard developed using R Shiny to analyze retail sales data from the Superstore dataset. The dashboard enables users to explore sales performance, profitability, customer behavior, and product trends through dynamic visualizations and filters.

# Objective
The objective of this project was to transform raw retail sales data into meaningful business insights using interactive dashboards. It helps users monitor key performance indicators (KPIs), analyze sales trends, evaluate product performance, and forecast future sales.

# Features
* Interactive dashboard built using R Shiny
* Dynamic filters for Region, Category, Segment, and Date Range
* KPI Cards displaying:
  * Total Sales
  * Total Profit
  * Total Orders
  * Unique Customers
* Sales trend analysis
* Regional sales performance
* Category-wise profitability analysis
* Customer segment analysis
* Top-selling products
* Correlation analysis
* 12-month sales forecasting using ARIMA

# Key Insights
* Identified the highest-performing regions based on sales and profit.
* Compared profitability across different product categories.
* Analyzed customer segments contributing the highest revenue.
* Identified top-selling products and sales trends over time.
* Forecasted future sales to support business planning.
